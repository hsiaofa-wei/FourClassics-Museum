# FourClassics-Museum · 四大名著数字博物馆

![项目封面](img/HomeImg/book.jpg)

一个基于纯前端技术（HTML + CSS + JavaScript）的四大名著互动展示平台，通过现代网页动画与交互设计，让经典文学“活”起来。本项目获奖**2025 永创杯第二届全国网页设计大赛**

## 项目简介
数字博物馆以四大名著（《红楼梦》《西游记》《水浒传》《三国演义》）为核心，采用纯静态网页形式，结合 CSS3 动画、JavaScript 交互与响应式布局，打造沉浸式文学阅读体验。

核心目标：
- 传承中国古典文学
- 展示前端交互设计能力
- 探索静态网站的高级动画与用户体验

项目完全独立完成，从创意构思到代码实现、参赛投稿一气呵成。

## 核心技术与亮点
- **纯静态架构**：零后端、无框架依赖，仅 HTML + CSS + JavaScript
- **高级交互**：
  - CSS3 动画（翻书、人物出场、场景切换、粒子效果）
  - JavaScript 事件驱动（点击、滚动、hover 触发动画）
  - 平滑过渡与视差滚动
- **响应式设计**：完美适配 PC、平板、手机
- **模块化开发**：代码结构清晰，易维护
- **性能优化**：图片懒加载、资源压缩、首屏速度快
- **获奖认证**：2025 永创杯全国网页设计大赛 三等奖

## 项目特点
- 四大名著专题页：每部名著独立模块，人物、情节、名句互动展示
- 沉浸式阅读：作者趣味对话、章节切换、背景音乐（可选）
- 文化元素融合：中国风配色、古籍纹理、书法字体
- 无障碍访问：语义化标签 + 键盘导航支持
